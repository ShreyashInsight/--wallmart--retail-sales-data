# 🛒 Walmart Retail Sales Dashboard

## 📊 Project Overview
An interactive Power BI dashboard analyzing Walmart 
retail sales data to uncover trends in customer 
behavior, product performance, and transaction patterns.

## 📁 Dataset
- **Source:** Walmart Retail Sales Data
- **Records:** 5000 Transactions
- **Customers:** 200
- **Products:** 50 unique items

## 🔍 Key Insights
- Top selling product: P037 (115 units)
- Highest sales year: 2024 (2.5K transactions)
- Most popular categories: Television, Clothing, Laptop

## 🛠️ Tools Used
- Microsoft Power BI
- Microsoft Excel
- DAX (Data Analysis Expressions)

## 📌 Dashboard Features
- KPI Cards (Customers, Quantity, Transactions)
- Sales trend over time (2023–2025)
- Product category breakdown
- Payment analysis by year
